# Project Brief: GhostNodes

## 🎯 Mission
To empower individuals with total digital sovereignty by providing a turn-key, secure, and lightweight orchestration layer for personal servers and Bitcoin nodes.

## 👁 Vision
A world where personal data and financial freedom are hosted on local, sovereign hardware, bypassing centralized cloud providers without sacrificing professional reliability or ease of use.

## 🚀 Strategic Goals
1. **Low Friction Deployment**: A single command should turn a bare-metal ARM64 board into a functional server.
2. **Deterministic Architecture**: Use Spec-Driven Development (SDD) to ensure the code always matches the architectural design.
3. **Hardware Efficiency**: Optimized for low-power devices (Orange Pi Zero series, Raspberry Pi), minimizing RAM and CPU overhead.
4. **Security by Default**: Every node is hardened automatically, including firewall rules and secure service isolation.

## 👤 Target Audience
- **Bitcoiners**: Individuals looking to run pruned or full nodes at home.
- **Privacy Enthusiasts**: Users wanting to self-host Pi-hole, VPNs, and file syncing.
- **Sovereign Tech Operators**: Developers building local-first infrastructure.
